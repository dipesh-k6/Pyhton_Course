\# Status codes 



&#x09;Code				Meaning 

&#x09;200				✅OK — success

&#x09;201				✅ Created — new resource created

&#x09;301				↪️ Redirect — moved permanently

&#x09;400				❌ Bad Request — client sent wrong data

&#x09;401				❌ Unauthorized — not logged in

&#x09;403				❌ Forbidden — logged in but no permission

&#x09;404				❌ Not Found — resource doesn't exist

&#x09;500				❌ Server Error — something broke on server
